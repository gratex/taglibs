Html tags

Standard Syntax:
     <%@ taglib prefix="h" uri="http://gratex.com/gjaxXB/tags/html" %>
XML Syntax:
     <anyxmlelement xmlns:h="http://gratex.com/gjaxXB/tags/html" />
 
Tags that are responsible for generating HTML or XHTML output, DocType and also setting appropriate Content-type. 

<DOCTYPE> tag MUST be used before any other tag! 



Configuration 

All settings can be done in four different ways: setting attributes of <DOCTYPE> setting init parameters (web.xml); setting servletContext attribute ; setting request attribute. If one parameter is set in more places preferences are: 1. request attribute 2. tag attribute 3. servletContext attribute 4. init parameter 5. defaul value

 

Example of setting init parameter: 

<context-param> 

<param-name>com.gratex.gjaxXB.tags.html.doctype</param-name> 

<param-value>STRICT</param-value> 

</context-param> 


 

Markup; 

KEY: com.gratex.gjaxXB.tags.html.markup;

VALUES:

    

  • HTML4 (default) - ContentType will be set to 'text/html'
    • 

  • XHTML - ContentType will be set according to 'xhtmlMode'
    • 

  • HTML5 - ContentType will be set to 'text/html'
    • 

  • XHTML5 - ContentType will be set according to 'xhtmlMode'
    • 



Omit end tags 

KEY: com.gratex.gjaxXB.tags.html.omitEndTags

When set to 'true' (if markup=='XHTML' it will be ignored), omits end tag in these tags: <DD> <DT> <LI> <OPTION> <TD> <TH> <TR&gt;

VALUES:

    

  • false (default)
    • 

  • true
    • 



XHTML mode 

KEY: com.gratex.gjaxXB.tags.html.xhtmlMode

This setting will be ignored if markup!='XHTML' or 'XHTML5'.

VALUES:

    

  • HTML_COMPATIBLE (default) - only EMPTY tags will be minimized, ContentType will be set to 'text/html'
    • 

  • FULL - no tags will be minimized, ContentType will be set to 'application/xhtml+xml'
    • 

  • MINIMIZED - tags that can be minimized and have no body will be minimized, ContentType will be set to 'application/xhtml+xml'
    • 



DTD 

KEY: com.gratex.gjaxXB.tags.html.dtd

Defines what DOCTYPE declaration will be printed out. If markup is 'HTML5' and DTD not set to 'NONE', <!DOCTYPE HTML> will be printed out.

VALUES:

    

  • STRICT (default)
    • 

  • TRANSATIONAL
    • 

  • FRAMESET
    • 

  • NONE
    • 



Encoding 

KEY: com.gratex.gjaxXB.tags.html.encoding

Default value is 'UTF-8'

   


Generate XML declaration 

KEY: com.gratex.gjaxXB.tags.html.generateXmlDeclaration

When set to 'true' (and markup=='XHTML' otherwise ignored), generates XML declaration (<?xml .. ?&gt;)

VALUES:

    

  • false (default)
    • 

  • true
    • 




Generate META tag 

KEY: com.gratex.gjaxXB.tags.html.generateMetaTag

When set to 'true' correct metatag with content type and encoding will be generated(<?xml .. ?>)

VALUES:

    

  • true (default)
    • 

  • false
    •

Attributes which value is NULL or empty String will be printed out.

All tags have additional attributes innerText and innerHTML. If one of these attribute is present, its content will be printed out instead of the body. HTML escaped if innerText and unescaped if innerHTML.

DOCTYPETag that is responsible for generating doctype, xml declaration, setting contentType. It MUST be used before any other tag! If more than one presence of DOCTYPE tag is found (for example nesting with Tiles), second and next usage will be ignored (and logged as Warning)
_HTML comment. Comment is printed to client code only if init parameter with key 'com.gratex.gjaxXB.tags.html.commentsOutput' is set to true.
If attribute 'content' is not null, body of tag is ignored. Content of comment is HTML escaped. This behaviour can be changed by init parameter 'com.gratex.gjaxXB.tags.html.commentsEscape' or attribute htmlEscape.
_ifIE conditional comment..
If attribute 'content' is not null, body of tag is ignored.
Content of comment is HTML escaped. This behaviour can be changed by init parameter 'com.gratex.gjaxXB.tags.html.commentsEscape' or attribute htmlEscape.

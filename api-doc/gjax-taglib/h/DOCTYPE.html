Html tags
Tag DOCTYPE

Tag that is responsible for generating doctype, xml declaration, setting contentType. It MUST be used before any other tag! If more than one presence of DOCTYPE tag is found (for example nesting with Tiles), second and next usage will be ignored (and logged as Warning)

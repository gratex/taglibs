Tag Summary
rootEnables using EL functions ('message', 'rr') from 'sh' namespace ('http://gratex.com/spring/tags/helpers').
If you want to use it instead of jsp:root, remember that you will have to write <jsp:output omit-xml-declaration="yes"/> which is default in jsp:root.
aA tag with URL tag support for href. (Like spring:url)
resourceResource. MUST be used as child of <scriptCombo> and <cssLinkCombo>
scriptComboGenerates script tag for combinated resources.
cssLinkComboGenerates link tag for css combinated resources.
evalEvaluates a Spring expression (SpEL) and either prints the result or assigns it to a variable.

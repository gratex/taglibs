Tag Column

Creates a column (on TH tag) in the grid with default formatter (formatter="defaultFormatter"). For activation this feature the user must insert require "gjax/formatters". Then user must insert grid.set("formatterScope", formatters); statement before grid.setStore(store); into the JavaScript file. Use within EnhancedGrid tag. Grids with multirow headers are not supported using taglibrary. Use original markup with table, thead, tr and th.

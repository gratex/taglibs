Dojo tags

Standard Syntax:
     <%@ taglib prefix="d" uri="http://gratex.com/gjax/tags/dojo" %>
XML Syntax:
     <anyxmlelement xmlns:d="http://gratex.com/gjax/tags/dojo" />
 
Dojo tags for layout, forms and grids (Widgets). <br />Form widgets (ValidationTextBox, DateTextBox, FilteringSelect, etc.) automatically generate ID attributes if they are required  <br />and not specified. ID attribute is required if 'label' or 'labelCode' attribute is specified. Generaded IDs are strings that consists <br />of a prefix and a number. Prefix is '_' if markup mode is HTML5, or 'gen' if not. Number is an integer (unique per response) <br />starting from 1 (_1, _2, _3, ...).

Tag Summary
ValidationTextBoxCreates a ValidationTextBox widget (on INPUT tag). The ValidationTextBox widget is a <input> element that checks if user’s input is valid (dijit/form/ValidationTextBox).
DateTextBoxCreates a DateTextBox widget (on INPUT tag). DateTextBox widget is date entry control that allows either typing or choosing a date from calendar widget (dojox/form/DateTextBox).
NumberTextBoxCreates a NumberTextBox widget (on INPUT tag). The NumberTextBox widget is a textbox for entering and displaying numbers. The NumberTextBox widget is used when user input is restricted to numeric input. (dijit/form/NumberTextBox).
FormCreates a Form widget (on FORM tag). The Form widget is equivalent of the HTML <form> element (dijit/form/Form)
CheckBoxCreates a Checkbox widget (on INPUT tag). Css class "rc-label" is added to label. Checkboxes are on/off switches that may be toggled by the user. A switch is "on" when the control element’s selected attribute is set. Several checkboxes in a form may share the same control name. The <input> element is used to create a checkbox control (dijit/form/CheckBox).
RadioButtonCreates a RadioButton widget (on INPUT tag). Css class "rc-label" is added to label Radio buttons are on/off switches that may be toggled by the user. User can select only one choice from a list. Several radio buttons in a form may share the same control name. The <input> element is used to create a radio button control. (dijit/form/RadioButton).
FilteringSelectCreates a FilteringSelect widget (on INPUT tag). The FilteringSelect widget is an enhanced version of the <select> element with type-ahead filtering of the drop down list. The FilteringSelect widget searches on the name attribute of objects in dojo.data store. When user open a FilteringSelect menu, by default will see the name attribute of each item in the menu. You can enrich the menu’s content by supplying a different label. (dijit/form/FilteringSelect).
ButtonCreates a Button widget (on BUTTON tag). The Button widget is a representation of a normal <button> with type="submit | reset | button". If not specified, type attribute defaults to 'button' (for browsers consistency). Buttons can have icons and labels (dijit/form/Button).
ItemFileReadStoreThe ItemFileReadStore (on DIV tag) implements the dojo.data.api. The ItemFileReadStore reads the JSON structured contents from an http endpoint (service or URL), or from an in-memory JavaScript object, and stores all the items in-memory for simple and quick access. (dojo/data/ItemFileReadStore).
AccordionContainerCreates a AccordionContainer widget (on DIV tag). The AccordionContainer widget holds a set of panes whose titles are all visible, but only one pane's content is visible at a time. (dijit/layout/AccordionContainer).
BorderContainerCreates a BorderContainer widget (on DIV tag) as a box with a specified size. The BorderContainer widget is a container partitioned into up to five regions: left, right, top, and bottom with a mandatory center to fill in any remaining space. (dijit/layout/BorderContainerTag).
ContentPaneCreates a ContentPane widget (on DIV tag). The ContentPane widget embeds a document fragment in the page, specified either by uri, javascript generated markup or DOM reference (dijit/layout/ContentPane).
StackContainerCreates a StackContainer widget (on DIV tag). The Stackcontainer widget is container for wizards, slide shows, and long lists or text blocks. (dijit/layout/StackContainer).
TabContainerCreates a TabContainer widget (on DIV tag). The TabContainer widget is a container with multiple panes, but shows only one pane at a time. (dijit/layout/TabContainer).
TitlePaneCreates a TitlePane widget (on DIV tag). The TitlePane widget is an accessible container with a title Heading, and a content section that slides open and closed. (dijit/TitlePane).
EnhancedGridCreates an EnhancedGrid widget (on TABLE tag). Output: <table> element, <thead> element and <tr> element. Grids with multirow headers are not supported using taglibrary. Use original markup with table, thead, tr and th. The EnhancedGrid widget is mini spreadsheet, commonly used to display details on master-detail forms. Use column tag to add fields (dojox/grid/EnhancedGrid).
ColumnCreates a column (on TH tag) in the grid with default formatter (formatter="defaultFormatter"). For activation this feature the user must insert require "gjax/formatters". Then user must insert grid.set("formatterScope", formatters); statement before grid.setStore(store); into the JavaScript file. Use within EnhancedGrid tag. Grids with multirow headers are not supported using taglibrary. Use original markup with table, thead, tr and th.
dojo.jsLoads dojo.js with dojo configuration.
NumberSpinnerCreates a NumberSpinner widget (on INPUT tag). A dijit.form.NumberTextBox extension to provide keyboard accessible value selection as well as icons for spinning direction. When using the keyboard, the typematic rules apply, meaning holding the key will gradually increase or decrease the value and accelerate.
ComboBoxCreates a ComboBox widget (on SELECT tag). The ComboBox is a hybrid between a SELECT combo-box and an INPUT text field.
ComboButtonCreates a ComboButton widget (on BUTTON tag). ComboButton is a combination button and drop-down button. Users can click one side to "press" the button, or click an arrow icon to display the drop down.
DropDownButtonCreates a DropDownButton widget (on BUTTON tag). DropDownButton is a button with a drop down.
TextareaCreates a Textarea widget (on TEXTAREA tag). Textarea dynamically expands/contracts (changing it's height) as the user types, to display all the text without requiring a scroll bar. Takes nearly all the parameters (name, value, etc.) that a vanilla textarea takes. Rows is not supported since this widget adjusts the height.
SimpleTextareaCreates a SimpleTextarea widget (on TEXTAREA tag). SimpleTextarea is a simple textarea that degrades, and responds to minimal LayoutContainer usage, and works with dijit.form.Form. Doesn't automatically size according to input, like Textarea.
SelectCreates a Select widget (on SELECT tag). Select is a "styleable" select box - it is basically a DropDownButton which can take a <select> as its input.
TextBoxCreates a TextBox widget (on INPUT tag). TextBox is a base class for textbox form inputs.
TimeTextBoxCreates a TimeTextBox widget (on INPUT tag). TimeTextBox is a validating, serializable, range-bound time text box with a drop down time picker
ToggleButtonCreates a ToggleButton widget (on BUTTON tag). ToggleButton is a templated button widget that can be in two states (checked or not). Can be base class for things like tabs or checkbox or radio buttons
LinkPaneCreates a LinkPane widget (on A tag). LinkPane is just a ContentPane that is declared in markup similarly to an anchor. The anchor's body (the words between <a> and </a>) become the title of the widget (used for TabContainer, AccordionContainer, etc.)
MenuItemCreates a MenuItem widget (on DIV tag). MenuItem is a line item in a Menu Widget.
TooltipDialogCreates a TooltipDialog widget (on DIV tag). TooltipDialog pops up a dialog that appears like a Tooltip.
EditorCreates a Editor widget (on DIV tag). Editor provides basic WYSIWYG editing features, based on the browser's underlying rich text editing capability, accompanied by a toolbar (dijit.Toolbar). A plugin model is available to extend the editor's capabilities as well as the the options available in the toolbar. Content generation may vary across browsers, and clipboard operations may have different results, to name a few limitations. Note: this widget should not be used with the HTML <TEXTAREA> tag - see dijit._editor.RichText for details.
DropDownMenuCreates a DropDownMenu widget (on DIV tag). DropDownMenu is a menu, without features for context menu (Meaning, drop down menu).
DialogCreates a Dialog widget (on DIV tag). Dijit's modal Dialog Box simulates a regular GUI dialog box. The contents can be arbitrary HTML, but are most often a form or a short paragraph. The user can close the dialog box without acting by clicking on the X button in the top-right corner.
xDateTextBoxCreates a dojox/form/DateTextBox widget (on INPUT tag), a validating, serializable, range-bound date text box with a popup calendar.
CheckedMultiSelectCreates a dojox/form/CheckedMultiSelect widget (on SELECT tag). CheckedMultiSelect uses check boxes and provides a more user-friendly mechanism for multi-selects. By not specifying “multiple=true” in your widget, you will create a widget using radio buttons instead of check boxes.
UploaderCreates a dojox/form/Uploader widget (on INPUT tag). The widget adds functionality to a standard HTML file input which allows file uploading to a server. If the browser supports the HTML5 file input specification, that functionality is used. If it is not supported (IE or older browsers) plugins are available to enhance the Uploader.
FileListCreates a dojox/form/uploader/FileList widget (on DIV tag). It automatically connects to an Uploader and detects when files are added, removed, or uploaded and provides a list of the files currently selected by dojox.form.Uploader.
FloatingPaneCreates a dojox/layout/FloatingPane widget (on DIV tag). Floating panes can be resized, minimized/maximized and layered dynamically, similar to application windows within the web page. Unlike dijit.Dialog, FloatingPane is not modal and it is designed to allow multiple active windows.
RepeatCreates a dojox/mvc/Repeat widget (on DIV tag). A repeat is bound to an intermediate dojo.Stateful node corresponding to an array in the data model. Child dijits or custom view components inside it inherit their parent data binding context from it.
GroupCreates a dojox/mvc/Group widget (on DIV tag), a simple model-bound container widget with single-node binding to a data model.

Tag FloatingPane

Creates a dojox/layout/FloatingPane widget (on DIV tag). Floating panes can be resized, minimized/maximized and layered dynamically, similar to application windows within the web page. Unlike dijit.Dialog, FloatingPane is not modal and it is designed to allow multiple active windows.

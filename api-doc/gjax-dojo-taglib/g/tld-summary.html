Tag Summary
FilteringSelectCreates an extended FilteringSelect widget. This widget allows to set "expanded" value: {id:'1',name:'EN'} and deffers fetching store data until user clicks the widget. This tag The FilteringSelect widget is an enhanced version of the <select> element with type-ahead filtering of the drop down list. The FilteringSelect widget searches on the name attribute of objects in dojo.data store. When user open a FilteringSelect menu, by default will see the name attribute of each item in the menu. You can enrich the menuâ€™s content by supplying a different label. (gjax.form.FilteringSelect)
CurrencyTextBoxCreates a gjax/form/CurrencyTextBox widget. CurrencyTextBox widgets inherit all the attributes and behaviors of the NumberTextBox widget, but are specialized for input monetary values.
OutputCreates a gjax/Output widget.
LinkCreates a gjax/Link widget (on A tag) that enables full navigation control.
LinkButtonCreates a gjax/LinkButton widget (on BUTTON tag) that enables full navigation control.
ResetableTextBoxCreates a gjax/form/ResetableTextBox widget (on INPUT tag). Allows user to delete input text.
FieldsetCreates a gjax/Fieldset widget (on FIELDSET tag). A fieldset that can be expanded, collapsed or closed.
ValidationTextareaCreates a gjax/form/ValidationTextarea widget (on INPUT tag). Uses validation similar to ValidationTextBox.
